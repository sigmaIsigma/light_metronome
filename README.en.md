# Light Metronome

A clean, minimalist metronome web application featuring high-precision Web Audio API timing, subdivision support, individual volume controls, and an "Unsafe Mode" for professional use. Supports PWA for installation on mobile and desktop.

## Features

- **High-Precision Timing**: Utilizes Web Audio API's `audioContext.currentTime` for rock-solid rhythm independent of browser UI load.
- **Advanced Rhythm Settings**:
  - **Subdivisions**: Divide each beat into up to 16 parts (unlimited in Unsafe Mode).
  - **Visual Meter**: A grid-based display showing the current beat and subdivision index (1₁, 1₂...).
  - **Individual Volume**: Set specific volume percentages for the **Accent** (1st beat), **Beat** (other beat heads), and **Subdivisions**.
- **Flexible Controls**:
  - **Real-time Updates**: BPM and rhythm settings are applied instantly with an immediate restart from the first beat.
  - **Tap Tempo**: Calculate BPM automatically by tapping a button.
  - **Decimal BPM**: Supports precise tempo adjustment in 0.1 increments.
- **Dynamic Layout**:
  - Draggable divider to adjust the size of the display and control panes.
  - Smart-Scroll: Automatically keeps the current beat centered even with extreme beat counts.
- **Unsafe Mode**: Unlock standard limits (BPM 1000, Beats 64, Subdiv 16) for extreme rhythmic experimentation.
- **PWA Support**: Installable as a standalone app on supported browsers for offline use.

## Usage

Simply host the files on any static web server or open [this page](https://sigmaisigma.github.io/light_metronome/) in your browser.

1. Adjust BPM, Beats, and Subdivisions in the settings panel.
2. Click the **START** button to begin.
3. Any changes made while running will immediately restart the cycle from the first beat.
4. Click **STOP** to completely reset the metronome.

## Requirements

- A modern web browser supporting Web Audio API.
- JavaScript enabled.
